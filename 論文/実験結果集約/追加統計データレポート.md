# 追加統計データレポート

## 1. 順位一致の定量根拠（Spearman 順位相関）

### 概要

SBERT 類似度と LLM 評価の順位がどの程度一致しているかを定量化するため、Spearman 順位相関を計算しました。

### 結果

#### 全体（全 36 条件）\*\*\*\*

- **Spearman 順位相関係数**: ρ = 0.7266
- **p 値**: p < 0.001（統計的に有意）
- **サンプル数**: n = 36

#### データセット別

| データセット | Spearman 順位相関 | p 値      | サンプル数 |
| ------------ | ----------------- | --------- | ---------- |
| SemEval-2014 | 0.2582            | 0.7418    | 4          |
| GoEmotions   | 0.7677            | p < 0.001 | 28         |
| Steam        | -0.2582           | 0.7418    | 4          |

### 順位一致の詳細

#### 上位 3 位の一致

- **SBERT 類似度上位 3 位**: embarrassment, gratitude, disgust
- **LLM 評価上位 3 位**: disgust, embarrassment, joy
- **一致数**: 2/3（embarrassment, disgust）

#### 下位 3 位の一致

- **SBERT 類似度下位 3 位**: visual, story, neutral
- **LLM 評価下位 3 位**: admiration, approval, caring
- **一致数**: 0/3

### 解釈

- 全体では、SBERT 類似度と LLM 評価の順位に強い正の相関（ρ = 0.73）が確認された
- GoEmotions では特に高い順位相関（ρ = 0.77）を示し、データセット内での順位が概ね一致している
- SemEval と Steam ではサンプル数が少ない（n=4）ため、統計的有意性は低い

### 論文への引用例

> SBERT 類似度と LLM 評価は、スコアの水準は異なるものの、データセット間の順位が一致しており（Spearman 順位相関 ρ = 0.73, p < 0.001）、どの条件が相対的に良いかという傾向を概ね共有している。

## 2. Steam データセットの定量統計

### 概要

「Steam は長く雑多」という主張の根拠を定量化するため、Steam データセットのレビュー長、語彙多様性、アスペクト統計を計算しました。

### 結果

#### 基本統計

- **総レビュー数**: 1,100 件
- **平均文字数**: 1,367 文字（中央値: 871 文字）
- **平均語数**: 251 語（中央値: 163 語）
- **平均文数**: 14.3 文（中央値: 9 文）

#### レビュー長の分布

| 指標     | 文字数 | 語数  | 文数 |
| -------- | ------ | ----- | ---- |
| 平均     | 1,367  | 251   | 14.3 |
| 中央値   | 871    | 163   | 9.0  |
| Q1       | 412    | 74    | 4.0  |
| Q3       | 1,725  | 323   | 18.0 |
| 最小     | 15     | 3     | 1    |
| 最大     | 7,992  | 1,521 | 114  |
| 標準偏差 | 1,463  | 266   | 15.3 |

#### 語彙多様性

- **総トークン数**: 276,151 語
- **ユニーク語彙数**: 15,740 語
- **語彙タイプ/トークン比（TTR）**: 0.0570

#### 頻出語トップ 10

| 順位 | 語   | 出現回数 |
| ---- | ---- | -------- |
| 1    | the  | 13,181   |
| 2    | and  | 7,537    |
| 3    | to   | 7,191    |
| 4    | a    | 6,919    |
| 5    | of   | 5,607    |
| 6    | i    | 5,328    |
| 7    | you  | 5,251    |
| 8    | it   | 4,934    |
| 9    | game | 4,605    |
| 10   | is   | 4,345    |

### 解釈

- Steam データセットのレビューは平均 251 語と長く、最大 1,521 語に達するものもある
- 語彙の多様性が高く（TTR = 0.057）、15,740 種類のユニーク語彙が使用されている
- レビュー長の分散が大きく（標準偏差 266 語）、「長く雑多」という特性が定量化された

### 論文への引用例

> 一方で Steam はレビューが長く（平均 251 語、最大 1,521 語）雑多で（語彙タイプ数 15,740、TTR 0.057）、皮肉や複合評価などのノイズも大きいため、アスペクトがレビューの文脈に依存しやすい。

## 3. データセット間の統計的差分

### 概要

グループ A/B 間の統計的差分の大きさを定量化するため、KL divergence や語彙重複率を計算しました。

### 結果

データ構造の問題により、一部のデータセットでグループ分割が正しく実行できませんでした。これは、データセットのラベル形式が実験設定と異なる可能性があります。

### 今後の対応

- データセットのラベル形式を確認し、グループ分割ロジックを修正する必要があります
- 実験で実際に使用されたグループ分割方法を確認し、それに合わせて統計を計算する必要があります
