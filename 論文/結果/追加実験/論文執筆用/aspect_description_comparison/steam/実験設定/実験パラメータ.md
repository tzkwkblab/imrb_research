# アスペクト説明文比較実験パラメータ

## 実験の目的

アスペクト説明文の有無が対比因子抽出の性能に与える影響を検証するため、説明文ありとなしの対比実験を実施。

## 実験設定

| パラメータ | 値 |
|-----------|-----|
| **データセット** | Steam |
| **アスペクト** | gameplay, visual, story, audio（4アスペクト） |
| **GPTモデル** | gpt-4o |
| **group_size** | 100 |
| **temperature** | 0.0 |
| **max_tokens** | 2000 |
| **Few-shot** | 0-shot（固定） |
| **アスペクト説明文** | あり/なし（対比検証） |
| **LLM評価** | 有効（gpt-4o-mini, temperature=0.0） |
| **分割タイプ** | aspect_vs_others |
| **総実験数** | 8実験（4アスペクト × 説明文あり/なし） |
| **実行日時** | 2025-11-27 |

## 実験構成

各アスペクトについて、説明文なしと説明文ありの2パターンで実験を実施：

| 実験ID | アスペクト | アスペクト説明文 |
|--------|----------|----------------|
| `steam_gameplay_0_4o_no_desc` | gameplay | なし |
| `steam_gameplay_0_4o_with_desc` | gameplay | あり |
| `steam_visual_0_4o_no_desc` | visual | なし |
| `steam_visual_0_4o_with_desc` | visual | あり |
| `steam_story_0_4o_no_desc` | story | なし |
| `steam_story_0_4o_with_desc` | story | あり |
| `steam_audio_0_4o_no_desc` | audio | なし |
| `steam_audio_0_4o_with_desc` | audio | あり |

## 評価指標

- **BERTスコア**: 意味的類似度（主要指標）
- **BLEUスコア**: n-gram一致率（参考指標）
- **LLMスコア**: GPT-4o-miniによる意味的類似度評価（補助指標）

## アスペクト説明文の設定

- **説明文ファイル**: `data/analysis-workspace/aspect_descriptions/steam/descriptions_official.csv`
- **説明文ありの場合**: アスペクト名を説明文に置き換えてプロンプトに含める
- **説明文なしの場合**: アスペクト名をそのまま使用
