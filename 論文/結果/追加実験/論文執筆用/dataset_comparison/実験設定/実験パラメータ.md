# データセット別性能比較実験 - 実験パラメータ

## 実験概要

- **実験名**: データセット別性能比較実験
- **実行日時**: 2025-11-27
- **目的**: Steam、SemEval、GoEmotionsの3つのデータセット間で、0-shot設定での対比因子抽出性能を比較

## 実験設定

| パラメータ | 値 |
|-----------|-----|
| データセット | Steam (4アスペクト), SemEval (4アスペクト), GoEmotions (28アスペクト) |
| Few-shot設定 | 0-shot（固定） |
| GPTモデル | gpt-4o-mini |
| group_size | 100 |
| max_tokens | 2000 |
| temperature | 0.0 |
| use_llm_evaluation | True |
| LLM評価モデル | gpt-4o-mini |
| LLM評価温度 | 0.0 |
| use_aspect_descriptions | False |
| 分割タイプ | aspect_vs_others |

## 実験構成

- **総実験数**: 36実験
  - Steam: 4実験（gameplay, visual, story, audio）
  - SemEval: 4実験（food, service, battery, screen）
  - GoEmotions: 28実験（admiration, amusement, anger, annoyance, approval, caring, confusion, curiosity, desire, disappointment, disapproval, disgust, embarrassment, excitement, fear, gratitude, grief, joy, love, nervousness, neutral, optimism, pride, realization, relief, remorse, sadness, surprise）

## 評価指標

- **BERTスコア**: 意味的類似度（F1スコア）
- **BLEUスコア**: n-gram一致率
- **LLMスコア**: gpt-4o-miniによる意味的類似度評価（0.0-1.0）

## データセット詳細

### Steam
- **ドメイン**: ゲームレビュー
- **アスペクト**: gameplay, visual, story, audio
- **特徴**: ゲーム体験に関する主観的な評価

### SemEval
- **ドメイン**: Restaurant (food, service), Laptop (battery, screen)
- **アスペクト**: food, service, battery, screen
- **特徴**: 商品レビューにおける客観的な評価

### GoEmotions
- **ドメイン**: Redditコメント
- **アスペクト**: 28種類の感情（admiration, amusement, anger, etc.）
- **特徴**: 感情表現が豊富なテキストデータ
