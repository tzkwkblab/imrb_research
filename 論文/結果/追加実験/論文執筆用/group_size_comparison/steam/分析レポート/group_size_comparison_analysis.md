# Steam Group Size比較実験結果のLLM分析

生成日時: 2025-11-27 12:51:50

## 実験概要

- **データセット**: Steam
- **アスペクト**: gameplay, visual, story, audio（4アスペクト）
- **Group Size設定**: 50, 100, 150, 200, 300（5段階）
- **Few-shot**: 0
- **Temperature**: 0.0
- **Max tokens**: 2000
- **総実験数**: 20実験
- **成功**: 20実験
- **失敗**: 0実験

## 主要結果

### Group Size別の平均スコア
- **Group Size 50**: BERT=0.5489, LLM=0.2000
- **Group Size 100**: BERT=0.5514, LLM=0.2000
- **Group Size 150**: BERT=0.5406, LLM=0.2000
- **Group Size 200**: BERT=0.5487, LLM=0.2000
- **Group Size 300**: BERT=0.5603, LLM=0.2000

### アスペクト別の平均スコア
- **audio**: BERT=0.5500, LLM=0.2000
- **gameplay**: BERT=0.5525, LLM=0.3200
- **story**: BERT=0.5285, LLM=0.4000
- **visual**: BERT=0.5276, LLM=0.2000

## LLMによる考察

1. **Group Sizeによる性能変化の傾向**

BERTスコアは、Group Size 50〜300の間で0.540前後に収まりつつ、全体としてはわずかな上昇傾向が見られます（50: 0.5489 → 300: 0.5603）。150のみやや低下（0.5406）しているものの、変動幅は最大でも約0.02程度と小さく、**グループサイズによる極端な性能劣化は見られない**と言えます。LLMスコアは全Group Sizeで平均0.2000と一定であり、少なくとも本設定では**グループサイズによる人手（LLM）主観評価の差は検出できていません**。

BERTスコアが300で最も高いことから、**より大きなGroup Sizeは、ノイズを平均化し、A/Bの本質的差分を安定して抽出させる方向に働いている**可能性があります。ただし向上幅は限定的であり、「300が明確に最適」と言えるほどの決定的差ではありません。むしろ、50〜300の範囲であれば、**対比因子抽出の性能は概ねロバスト**であり、データ量増加のメリットは漸増的であると解釈できます。

2. **アスペクトによる性能差の要因**

アスペクト別では、BERT平均が gameplay (0.5525) ≒ audio (0.5500) > story (0.5285) ≒ visual (0.5276) となっており、**gameplay / audio の方が story / visual よりも意味類似度が高い**傾向があります。LLMスコアも同様に story (0.4000), gameplay (0.3200) が高く、audio, visual (各0.2000) が低い。

このパターンから、
- **gameplay / story**：ユーザーレビューにおける言語が比較的「説明的・抽象的」（例：バランス、操作性、ストーリー性、感情、コミュニティ）であり、LLMが抽出しやすい概念レベル差分を多く含む
- **visual / audio**：本来は具体的（グラフィック、サウンド）なアスペクトだが、本実験ではLLM出力が「ストーリー」「感情」「メカニクス」などに偏っており、**実際にニューロンが捉えている差分とアスペクト名（ground truthラベル）のミスマッチ**が大きい

と考えられます。実際、audioの対比因子出力はほぼすべて「ストーリー」「感情的体験」「メカニクス」中心であり、**音響的要素がほとんど言及されていない**ため、audio という正解アスペクトとの意味類似は構造的に低くなります。同様に visual でも「ユーモア」「キャラクター」「コミュニティ」など、視覚から離れた要素が多く、BERTスコア・LLMスコアが伸びにくくなっていると解釈できます。

また、各アスペクト内でのGroup Sizeによる変動幅は概ね0.02前後に収まり、アスペクトごとに「大きいGroup Sizeほど必ず高い」といった強い傾向はありません。これは、**アスペクト間の難易度差の方が、グループサイズ差よりも支配的**であることを示しています。

3. **評価指標間の関係性**

BLEUスコアが全ケースで0.0000であることから、**生成された対比因子ラベルは表層語彙レベルで正解アスペクト名とほぼ一致していない**ことが分かります。一方で、BERTスコアは0.50〜0.56程度を保っており、**語彙は違っても意味的にはある程度近接している**と評価されています。これは、本タスクにおいてBLEUが不適切であるという研究背景の主張を、実験的に裏付ける結果になっています。

LLMスコアは0.2〜0.6と粗い離散値であり、平均0.28とやや低めですが、story（0.40）、gameplay（0.32）が比較的高く、audio / visual（0.20）が低い点はBERTスコアと整合的です。したがって、
- **BERTスコアとLLMスコアは、アスペクト単位の難易度差の方向性に関しては概ね相関している**
- BLEUは完全に0であり、**意味的な評価に寄与していない**

という関係にあります。また、Group Sizeの変化に対して、
- BERTスコア：わずかな増減が検出される
- LLMスコア：平均値レベルでは変化せず（0.2000で固定）

という違いがあり、**LLMスコアはこの程度の差分（0.01〜0.02）では感度が足りない**と考えられます。より微細な差を評価するには、連続値かつ学習ベースの指標（BLEURT, BARTScore, MoverScore など）への移行が妥当です。

4. **研究への貢献**

本追加実験からは、以下の示唆が得られます。

- **LLMによる対比因子ラベル生成は、Group Size 50〜300の範囲で大きく破綻せずに動作し、意味類似度（BERT ≒0.54）は安定している**  
  → ある程度のサンプル数を与えれば、LLMはA/B集合の差分を一貫した抽象概念として言語化できる、という「実現可能性」の実証。

- **グループサイズ増加による性能向上は漸増的であり、300がわずかに有利だが、コストとのトレードオフを考えれば100〜200程度が実用的なバランス**と考えられる  
  → 大規模ニューロングラフ全体で多数ノードにラベルを振る実運用を想定すると、「最低50〜100あれば十分、可能なら200〜300で安定性向上」という指針になる。

- アスペクトによる性能差から、**LLMは「ストーリー・感情・メカニクス」といった抽象・内容面の差分を生成しやすく、audio / visualのような感覚的アスペクトでは、潜在概念と表層アスペクト名のギャップが評価を下げる**ことが分かった。  
  → これは、C-XAIで重要な「概念ベクトルの言語的命名は、必ずしも既存のラベル体系に一致しない」という事実を示し、「ニューラル概念に合わせて人間側のラベル体系を再設計する」必要性も示唆している。

総じて、この実験は、**LLMベース対比因子ラベリングは、ある程度のグループサイズで安定に機能し、特に抽象的な行動・内容アスペクトに対して有望である**こと、そして**評価にはBLEUではなく意味類似に敏感な学習ベース指標が必須である**ことを、定量的に裏付ける結果になっていると評価できる。
