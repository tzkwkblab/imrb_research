# 実験レポート：レビュー特徴数削減による判定精度向上実験

## 実験概要

- 実験日時：2025-05-08 16:35
- 実験者：清野駿
- 実験 ID：feature_reduction_001

## 目的

商品レビューの特徴判定において、特徴数を 20 個から 5 個に削減することで判定精度の向上を図る。また、特徴数の削減が分析の効率性と結果の解釈可能性に与える影響を検証する。

## 実験条件

- 使用モデル：gpt-4.1-mini
- パラメータ：
  - temperature: 0.7
  - 試行回数: 5 回（多数決方式）
- 特徴リストの要素数：5（従来の 20 から削減）
- 分析対象：商品レビューデータセット（300 件）

## プロンプト

```
各レビューに対して、5つの特徴（5features.csv）について0/1で判定するプロンプトを作成し、OpenAI API（gpt-4.1-mini）で判定を実施。
```

## 実験データ

- 入力データ：
  - ファイル：src/data/examples/sample_reviews_300.csv
  - 件数：300
  - 特徴：5features.csv に定義された 5 つの特徴
- 出力結果：
  - 保存先：src/analysis/results_batch_analysis/20250508_1635/
  - 形式：all_results.json, summary_report.md, 中間結果 json

## 実験結果

### 定量的分析

- 処理件数：300 件
- エラー率：0%（API エラー・中断なし）

### 特徴分析結果

- 特徴ごとの判定分布や安定性スコアは summary_report.md に記録（詳細な数値はここでは省略）
- 特徴数削減により、判定結果の分布が偏りすぎることなく、全特徴で一定のバリエーションが得られた

## 考察

### 成功点

- 特徴数を 5 つに絞ることで、判定プロセスがシンプルになり、出力の解釈も容易になった
- 全件で API エラー等の問題なく処理が完了した

### 問題点

- 現時点では人手による精度検証や詳細な誤判定分析は未実施
- 特徴数削減による判定精度の定量的な比較は今後の課題

### 仮説の検証

- 特徴数削減による判定効率・解釈性の向上は確認できたが、精度向上については今後人手検証が必要
