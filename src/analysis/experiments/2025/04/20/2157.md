# 実験レポート：300 件サンプルデータの特徴分析

## 実験概要

- 実験日時：2025-04-20 21:57
- 実験者：清野駿

## 人間評価実験

### 目的

GPT によって判定された 300 件のレビューサンプルに対する特徴判定の信頼性を、人間の目で直接検証します。

### 評価方法

1. 評価データの準備

   - all_results.json から評価用 md ファイルを自動生成
   - 各レビューに対して、20 個の特徴それぞれについて GPT の判定結果を表示
   - 人間が判定を確認し、不適切な判定にチェックを入れる形式

2. 評価フォーマット

```markdown
# レビュー ID: {review_id}

評価点: {rating}

## レビュー本文

{review_text}

## 特徴判定

特徴 1（価格やコストへの言及）：{0 or 1}
判定（不適切な場合チェック）

- [ ]

特徴 2（技術的な詳細や仕様への言及）：{0 or 1}
判定（不適切な場合チェック）

- [ ]

...
```

3. 評価手順

   - 各レビューのテキストを読む
   - 各特徴について、GPT の判定（0/1）が適切かを確認
   - 不適切と判断した場合、チェックボックスにチェックを入れる
   - 必要に応じてコメントを追記

4. 結果の集計方法
   - 特徴ごとの誤判定率の算出
   - 特に誤判定の多い特徴カテゴリの特定
   - 判定基準の改善が必要な項目の洗い出し

### 実装計画

1. 評価用 md ファイル生成スクリプトの作成

   - 入力：all_results.json
   - 出力：人間評価用の md ファイル
   - 実装場所：src/analysis/experiments/2025/04/20/

2. 評価実施

   - 生成された md ファイルを使用して人間評価を実施
   - 結果を随時保存

3. 結果分析
   - チェックされた項目の集計
   - 特徴ごとの誤判定率の算出
   - 改善提案の取りまとめ

### 人間で評価した結果

#### レビュー 001

- 特徴 5「具体的な使用例や目的を説明」：GPT の判定 0 を 1 に修正
  - 理由：複数の具体的な使用機器（コンピューター、mp3、ipod、ステレオシステム）との組み合わせについて説明があるため

#### レビュー 002

- 特徴 7「製品の制限や欠点について具体的に言及」：GPT の判定 1 を 0 に修正
  - 理由：製品自体の具体的な制限や欠点についての言及はなく、主にカスタマーサービスの問題について述べているため
- 特徴 12「製品の耐久性や品質について」：GPT の判定 1 を 0 に修正
  - 理由：不良品という言及はあるものの、具体的な品質や耐久性の問題についての説明がないため

#### レビュー 003

- 特徴 2「技術的な詳細や仕様を含む」：GPT の判定 0 を 1 に修正
  - 理由：画面表示の技術的な問題（コントラスト、文字の濃さの変化など）や操作インターフェースについて具体的な説明があるため
- 特徴 4「長期使用経験について」：GPT の判定 1 を 0 に修正
  - 理由：14 日以内の返品を予定しており、短期間の使用経験しか述べられていないため
- 特徴 17「カスタマーサポートの経験」：GPT の判定 1 を 0 に修正
  - 理由：カスタマーサービスについての言及はあるものの、具体的なサポート経験の記述がないため

### 間違い度合いの評価

#### 評価基準（5 段階）

1. ほぼ解釈の違い：異なる判断も十分に理解できる
2. 微妙な判断の相違：文脈によって判断が分かれる可能性がある
3. やや明確な間違い：文脈から判断すると誤りと言えるが、全く的外れではない
4. 明確な間違い：レビュー内容と明らかに矛盾している
5. 完全な間違い：レビューの内容を完全に誤解している

#### 各ミスの評価

##### レビュー 001

- 特徴 5「具体的な使用例や目的を説明」（GPT 判定：0→ 正解：1）
  - 間違い度合い：4（明確な間違い）
  - 理由：複数のデバイスでの具体的な使用例が明確に列挙されているにもかかわらず、0 と判定しているため

##### レビュー 002

- 特徴 7「製品の制限や欠点について具体的に言及」（GPT 判定：1→ 正解：0）

  - 間違い度合い：2（微妙な判断の相違）
  - 理由：カスタマーサービスの問題を製品の欠点として捉えることも可能で、文脈によって判断が分かれる可能性がある

- 特徴 12「製品の耐久性や品質について」（GPT 判定：1→ 正解：0）
  - 間違い度合い：3（やや明確な間違い）
  - 理由：不良品という言及はあるものの具体的な説明がないため、やや明確な間違いだが、品質に関する言及が全くないわけではない

##### レビュー 003

- 特徴 2「技術的な詳細や仕様を含む」（GPT 判定：0→ 正解：1）

  - 間違い度合い：4（明確な間違い）
  - 理由：画面のコントラストや文字の濃さの変化など、具体的な技術的問題について詳細に説明されているにもかかわらず、0 と判定している

- 特徴 4「長期使用経験について」（GPT 判定：1→ 正解：0）

  - 間違い度合い：5（完全な間違い）
  - 理由：14 日以内の返品を明確に述べており、長期使用とは完全に矛盾する内容である

- 特徴 17「カスタマーサポートの経験」（GPT 判定：1→ 正解：0）
  - 間違い度合い：2（微妙な判断の相違）
  - 理由：カスタマーサービスについての言及はあるが具体的な経験の記述がないという微妙な判断の相違
